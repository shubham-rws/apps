Requirements:

Node.js & NPM: https://nodejs.org/en/download/

Gulp: npm install gulp --save-dev (https://www.npmjs.com/package/gulp)

Gulp Sourcemaps: npm install gulp-sourcemaps --save-dev (https://www.npmjs.com/package/gulp-sourcemaps)

Gulp Sass: npm install gulp-sass --save-dev (https://www.npmjs.com/package/gulp-sass)

Gulp Autoprefixer: npm install --save-dev gulp-autoprefixer (https://www.npmjs.com/package/gulp-autoprefixer)

Gulp Concat: npm install gulp-concat --save-dev (https://www.npmjs.com/package/gulp-concat)

Gulp Uglify: npm install gulp-uglify --save-dev (https://www.npmjs.com/package/gulp-uglify)

NOT NEEDED, BUT LEAVING HERE FOR REFERENCE: Foundation CLI: npm install -g foundation-cli (https://github.com/zurb/foundation-cli)

After installing all of these in your repo theme directory via the command line you can then use "npm run build" and "npm run watch"
